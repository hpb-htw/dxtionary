# dxtionary README

A Plugin for Visual Studio Code to show a wiktionary entry.

## Features

* Show a wiktionary entry (Offline or Online)
* Lookup a wiktionary entry of a word from editor (Offline or Online)

## Requirements

* Internet connection for online mode
